## How to use

JS:

```
fabuloos("video", {});
```

Then, see the API site.

iOS:

* Import in XCode, then:

```
new Fabuloos()
```

Then, see the API site.

Android:

* Import in Eclipse, then:

```
new Fabuloos();
```

Then, see the API site.

W8:

* Import in Visual Studio, then:

```
new Fabuloos();
```

Then, see the API site.

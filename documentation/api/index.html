API

Fabuloos’ API is pretty straightforward.
Before you find the methods you need, here is some tips:

Most of the methods support the chaining, which means you can write something like this:

{% highlight js %} fab("player").src("/videos/clip.mp4").autoplay(true) {% endhighlight %}

Every properties have its own method to get or set its value. For example, .autoplay() will return the current autoplay value while .autoplay(true) will define the autoplay’s value.

And the most important: Have fun!

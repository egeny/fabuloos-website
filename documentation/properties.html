Properties

Every property have its own accessor method which you can call to get or set the value:

{% highlight js %} player.autoplay(); // Get the autoplay's value player.autoplay(true); // Set the autoplay's value {% endhighlight %} {% alert %} Some properties aren't correctly mapped in the renderers. Give us some time to fix this. {% endalert %}

Getters

autoplay Is the player launching the playback automatically?
buffered The buffered ranges of the media.
Currently badly implemented, we recommend to not use this property for now
controls Are the native controls displayed?
currentSrc The source currently playing, may be different from src.
currentTime The current position of the playback in seconds.
defaultPlaybackRate The default playback speed.
Currently badly implemented, we recommend to not use this property for now
duration The duration of the media.
Might be NaN if the source's metadata wasn't fetched yet.
Might be Infinity if you're trying to play a live stream.
ended Does the player ended the playback?
Will always be false if loop is true.
error The current error of the player.
null means no error.
Currently badly implemented, we recommend to not use this property for now
height The current height of the player.
loop Is the player have to start playing the media back when it ended?
muted Is the player currently muted?
networkState The current network state.
Currently badly implemented, we recommend to not use this property for now
paused Is the player currently paused?
playbackRate The current playback speed.
Currently badly implemented, we recommend to not use this property for now
played The range of seconds played.
Currently badly implemented, we recommend to not use this property for now
poster The poster image to use before launching the playback of the video.
preload The preloading mode.
Currently badly implemented, we recommend to not use this property for now
readyState The current player's state.
Currently badly implemented, we recommend to not use this property for now
seekable The range of seconds seekable.
Currently badly implemented, we recommend to not use this property for now
seeking Is the player currently seeking?
src The source currently playing.
videoHeight The height of the video's source.
videoWidth The width of the video's source.
volume The current volume between 0 and 1.
While muted the volume still have a value.
width The current width of the player.

Setters

autoplay Do the player launch the playback automatically?
controls true to display the native controls.
false to disable the controls.
currentTime Seek to a given position.
defaultPlaybackRate Set the default playback speed.
Currently badly implemented, we recommend to not use this property for now
height Set the height of the player.
loop Do the player have to start playing the media back when it ended?
muted Mute or unmute the player.
playbackRate Set the current playback speed.
Currently badly implemented, we recommend to not use this property for now
poster Set the poster image to use before launching the playback of the video.
preload Set the preloading mode.
Currently badly implemented, we recommend to not use this property for now
src Set the source to play.
volume Set the volume between 0 and 1.
While muted the volume still have a value.
width Set the width of the player.

Togglers

autoplay Toggle the autoplay's value.
controls Toggle the native or custom controls.
This is not really useful and may not work on every renderers.
loop Toggle the loop's value.
muted Toggle the muted value.
